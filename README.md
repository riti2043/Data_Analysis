# Data_Analysis
## Data Cleaning and Preprocessing with Pandas
### Overview

This project focuses on cleaning and preparing a dataset containing customer details and activity information. The goal was to identify and correct common data quality issues so the dataset becomes reliable for analysis or machine learning.

### Tasks Performed

* Loaded and explored the dataset using pandas

* Identified missing values in categorical and numerical columns

* Removed records with missing CustomerID

* Filled missing categorical values (Gender, Country, PreferredDevice)

* Filled numerical values such as Age and TotalPurchase

* Converted SignupDate and LastLogin to datetime format

* Corrected invalid entries (numeric characters in names, negative values)

* Standardized categorical values and fixed spelling inconsistencies

* Removed duplicate records

* Exported the cleaned dataset as a new CSV file
